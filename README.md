# QuizBattleApp

QuizBattleApp is a JavaFX-based quiz platform designed for educational environments. The application enables teachers to create and manage quizzes while students participate in interactive quiz battles. The system includes user-friendly interfaces, score tracking, and real-time quiz interactions to enhance the learning experience.

## Features
- Teacher login and quiz management
- Student participation interface
- Interactive quiz sessions
- Score calculation and leaderboard
- JavaFX graphical user interface
- Maven project structure

## Technologies Used
- Java
- JavaFX
- Maven
- FXML
- CSS
